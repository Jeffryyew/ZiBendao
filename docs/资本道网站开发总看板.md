## 1. 产品锚点（不可遗忘）

**使命**：Transform Businesses Into Investable Enterprises — 助力企业转型，成为值得投资的对象。

**核心定位**：资本道 **不是** ERP、CRM 或会计软件。它是引导企业逐步完成资本化的操作系统。

**核心用户旅程（产品脊柱）**
```
财务规范化  →  企业资本化  →  融资化  →  投资化
```

**设计哲学**：复杂逻辑隐藏在后台，用户永远只看到下一步。

**视觉风格**：Apple（简洁高级）+ Notion（结构清晰）+ Duolingo（成长式正反馈）。

## 3. 开发原则（每次开发前必读）

1. 不破坏现有功能。
2. 保持极简设计。
3. 优先提升用户体验。
4. 避免 ERP 与会计软件风格。
5. 所有功能围绕企业资本成长设计。
6. 用户永远只看到下一步。
7. 优先优化现有功能，而不是增加复杂功能。

**工作流**：分析架构 → 评估影响 → 提出方案 → 再修改。
**交付必附**：修改内容 · 测试结果 · 风险说明 · 下一步建议。

## 6. 税务与合规备忘（马来西亚，需本地专业人士确认）

- **电子发票（LHDN MyInvois）**：年营收 **< RM100 万目前豁免**；超过 RM100 万须接入。
  单笔交易 ≥ RM10,000 须开独立电子发票（2026 全年宽限，2027 起严格执行）。
- **SST（数字服务税）**：随支付上线一并确认。
- **PDPA**：处理企业/个人数据的硬性合规要求。
- 免责声明 / 投资建议表述：可能触及证监会（SC）范畴，建议律师过一遍。
> 本节为提醒方向，非法律意见。
